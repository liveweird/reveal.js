Working with Backbone.js

Backbone.js gives structure to web applications and helps to create rich client-side applications

Backbone...

  • offers the best when data is managed in JSON format
  • can be used extensively to create SPA
  • can also be used partially, in conjunction to server side generated HTML

Main elements are:

  • Model
  • View
  • Backbone.Events
  • Router
  • History
  • extend function

Backbone.Marionette

  • The extension to never go without
  • offers Layouts and Regions management
  • offers CollectionView and CompositeView objects
  • offers Application, Module and AppRouter
