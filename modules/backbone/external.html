Working with Backbone.js

Backbone.js gives structure to web applications and helps to create rich client-side applications

Backbone...

  • offers the best when data is managed in JSON format
  • can be used extensively to create SPA
  • can also be used partially, in conjunction to server side generated HTML

is inspired by MVC pattern

Main elements are:

  • Model
  • View
  • Router
  • History
  • Backbone.Events
  • extend function

Backbone.Marionette

  • The extension to never go without
  • offers Layouts and Regions management
  • offers CollectionView and CompositeView objects
  • offers Application, Module and AppRouter objects

The Design

Backbone library is designed in such a way that every single function can be overridden without modifying the original library, and the quality of the code is extreme.

  • extend function allows to easily define reusable objects
  • unlike other libraries, Backbone doesn't offer automatical two-ways binding
