Ripple

the Fubu alternative to NuGet

Environment

  • Highly modular app
  • Team City
  • Frequent module builds published to a gallery
  • Plenty of artifacts

Requirements

  • Handling artifacts published on regular basis
  • Not changing you project during updates
  • File/Nuget Gallery feeds handling (test purposes).

Package management for .NET

  • NuGet
  • OpenWrap (from OpenRasta)
  • Ripple (from Fubu)

Ripple to the rescue

  • Two kinds of dependencies: float / fixed
  • Dependencies extracted to directories with no version!
  • Versions declared on the solution ripple.config file level

Ripple codebase

  • Opensource: https://github.com/DarthFubuMVC/ripple
  • Part of the Fubu family
  • Very well tested
  • Fun to work with
